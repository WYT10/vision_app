## Dynamic mode keys
- `w / s` move both zones up/down
- `t / g` upper zone height + / -
- `y / 6` lower zone height + / -
- `i / k` move lower zone closer / farther
- `a / d` ROI width - / +
- `z / x` ROI height - / +
- `j / l` ROI gap above upper zone - / +
- `c / v` min red pixels - / +
- `f / b` min red ratio - / +
- `n / m` max x-difference - / +
- `[` / `]` pixel step down / up
- `TAB` toggle fixed / dynamic mode
